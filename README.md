Predict quote moderation decision
==========

## What's the goal?
The goal is to to predict the moderation decision for a submitted quote using a SVM.

**Currently in development.**

## Dependencies
- Python (>= 2.6 or >= 3.3)
- NumPy (>= 1.6.1)
- SciPy (>= 0.9)
